Как отредактировать *.bmp заставки конца игры.

  1. Откройте modes/2d_end_screen/vangers_pal_to_tga_pal.cfg, modes/2d_end_screen/usage_pal.cfg и modes/2d_end_screen/bmp_to_tga.cfg в любом текстовом редакторе. Замените значение опции source_dir на путь к папке data/resource/video/img Вангеров.
  2. Запустите modes/2d_end_screen/get_tga_screens.bat.
  3. Теперь папка modes/2d_end_screen/in_tga содержит *.tga изображения заставок, и их можно редактировать или заменять. Смотрите docs/common/2d_end_screen/end_screens.ods. Убедитесь, что все изображения в папке имеют разрешение 800x600, являются 8-битными и не используют RLE сжатие. Последние 16 цветов в палитре из 256 должны быть неиспользуемыми. То есть при конвертации в 8-битное изображение нужно использовать палитру из 240 цветов.
  4. Запустите modes/2d_end_screen/add_font_pal_to_screens.bat.
  5. Теперь папка modes/2d_end_screen/out_tga содержит *.tga изображения заставок с палитрой текста. Ее можно изменить, отредактировав последние 16 цветов палитры изображения.
  6. Запустите modes/2d_end_screen/tga_to_bmp.bat.
  7. Теперь папка modes/2d_end_screen/out_bmp содержит отредактированные *.bmp изображения и modes/2d_end_screen/out_pal/vangers/full содержит отредактированные *.pal файлы. Содержимое этих папок нужно скопировать в папку data/resource/video/img Вангеров. Стоит создать резервную копию оригинальных данных перед тем, как это делать.
